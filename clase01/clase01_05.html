El elemento <ul> crea una lista sin orden establecido. El tipo de marcador se puede variar: disc, circle, square o none.

Listas Ordenadas <ol>

El elemento <ol> crea una lista numerada. El atributo type define el estilo: 1, a, A, i, I. El atributo value permite establecer un número de inicio personalizado.
