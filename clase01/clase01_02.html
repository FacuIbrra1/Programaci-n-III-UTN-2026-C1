El elemento <p> - Párrafos

El elemento P representa un párrafo. Cada párrafo genera un bloque de texto separado del anterior y del siguiente.

Esta etiqueta produce un salto de línea y una línea en blanco entre párrafos.

Todo texto debe estar contenido dentro de un elemento apropiado, como este párrafo.

El elemento <hr> - Separador horizontal

El elemento HR produce una línea de separación entre secciones del documento.

Encabezados

Los encabezados van de <h1> (más importante) a <h6> (menos importante). Se deben usar respetando la jerarquía del contenido.
